Aider Bench Harness

SWE-bench Evaluation Pipeline: Task → Agent → Eval → Judge

Control Flags

  • • --skip-agent: Agent überspringen, direkt Eval
  • • --skip-eval: Eval überspringen, nur Agent
  • • --patch-path [NEW]: Eval mit bestehender Patch-Datei
  • • --synthetic-tests-json: Eigene Tests statt Eval
  • • --agent-timeout: Default 1800s
  • • --eval-timeout: Default 7200s

Artifacts (pro Run)

  • • git_diff.patch — Agent Output
  • • tests.json — Eval Ergebnis
  • • judge_input.json — Judge Input
  • • agent_stdout.log / stderr.log
  • • eval_stdout.log / stderr.log
  • • events.jsonl — Alle Phase-Events

Failure Reasons

  • • AGENT_TIMEOUT / AGENT_EXIT_NONZERO
  • • EVAL_TIMEOUT / EVAL_REPORT_MISSING
  • • EVAL_CONTAINER_CLEANUP_ERROR
  • • TASK_FAILURE (tests fehlgeschlagen)
  • • SUCCESS (alle Tests bestanden)
  • • derive_status → infra_failed | failed | success
